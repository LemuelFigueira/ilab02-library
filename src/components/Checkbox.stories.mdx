# Checkbox
O _**`<Checkbox>`**_ é um componente de botão que pode ser usado para ações de controle de estado.

## Props
| Prop | Type | Default | Description |
| :--- | :--- | :------ | :---------- |
| `children` | `ReactNode<string>` | `undefined` | Conteúdo do botão |
| `checked` | `boolean` | `false` | Define se o botão está selecionado ou não |
| `onCheckedChange` | `function` | `() => {}` | Função que será executada quando o botão for clicado |

<Meta title="Docs/Checkbox" component={ Checkbox } />

export const Template = (args) => <Checkbox {...args} />;
